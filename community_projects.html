Automate flux node installations using Ansible!

If you run a significant amount of Flux nodes, and/or if you use Ansible to manage them, this repo can really help by allowing the entire install to be handled by Ansible.

Previously Ansible struggled to fully automate the process because it had to run Multitoolbox for the installation/configuration process.

Now all flux node maintenance tasks, as well as new installations can be handled by a custom Ansible build.
